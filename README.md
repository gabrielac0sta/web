# Lista 1 de Programação para Internet
## Atividade 1
Desenvolvida para mostrar os conceitos básicos e implementar as principais funcionalidades das tags. Junto do programa vem a hitória do html e da web com contexto histórico e características. 
## Atividade 2
Explicações sobre o protocolo http e como ele é usado em web, atividade desenvolvida principalmente para a implementação de imagens e para o entendimento da contribuição entre clientes e recursos de servidores.
## Atividade 3
Mostra a criação de uma tabela em html desenvolvendo suas linhas e colunas e osganizando o conteúdo desejado na tabela final.
## Atividade 4
Usa a implementação de links, imagens, tabela e tags para reforçar os conteúdos tratados anteriormente. O formato de curriculo é bom para mostrar essas aplicações na prática.
