# Lista 1 de Programação para Internet
## Atividade 1 (história web)
Desenvolvida para mostrar os conceitos básicos e implementar as principais funcionalidades das tags. Junto do programa vem a história do HTML e da web com contexto histórico e características.
## Atividade 2 (protocolo http)
Explicações sobre o protocolo HTTP e como ele é usado na web. Atividade desenvolvida principalmente para a implementação de imagens e para o entendimento da comunicação entre clientes e servidores.
## Atividade 3 (tabela estados)
Mostra a criação de uma tabela em HTML, desenvolvendo suas linhas e colunas e organizando o conteúdo desejado na tabela final.
## Atividade 4 (curriculo)
Usa a implementação de links, imagens, tabelas e tags para reforçar os conteúdos tratados anteriormente. O formato de currículo é bom para mostrar essas aplicações na prática.
## Atividade 5 (contato)
Nessa atividade foram trabalhados os campos de texto no formato de label, utilizando a tag form com inputs, selects e options. Além disso, foi colocado ao final da página um link de navegação para o arquivo criado anteriormente (currículo).
## Atividade 6 (style)
Após todos os conceitos básicos de programação em HTML, foi adicionado um arquivo externo CSS com o intuito de organizar e melhorar a estética das páginas criadas, definindo as fontes e cores de texto, a cor do fundo e dos campos de label, as configurações da tabela (como espaçamento, borda e cor). Por fim, foi adicionado o seletor :hover para melhorar o uso de botões e links.
## Atividade final (entrega)
O repositório final foi adicionado ao GitHub e configurado no GitHub Pages para ter acesso à página inicial do projeto. Ao final, a atividade foi entregue por meio do link do GitHub.
