A Internet é a infraestrutura global de redes de computadores interconectadas.
A Web é um serviço que funciona sobre a Internet, permitindo acesso a páginas e conteúdos.
Em resumo: a Internet é a rede, a Web é um dos serviços.
A Web usa protocolos como HTTP para funcionar.
Sem Internet, não há Web.

História da Internet e da Web

A Internet surgiu nos anos 1960 com a ARPANET, criada nos EUA para conectar computadores militares e acadêmicos. Já a World Wide Web foi desenvolvida em 1989 por Tim Berners-Lee.
