O que é o protocolo HTTP?

O HTTP (HyperText Transfer Protocol) é o protocolo usado para comunicação entre navegadores e servidores na Web. Ele define como requisições e respostas devem ser estruturadas e transmitidas.

Exemplo de Requisição HTTP

            GET /index.html HTTP/1.1
            Host: www.exemplo.com
            User-Agent: Mozilla/5.0
            Accept: text/html
GET
Método da requisição (pode ser GET, POST, PUT, DELETE).
/index.html
Recurso solicitado no servidor.
Host
Indica o domínio do servidor.
User-Agent
Identifica o navegador/cliente que faz a requisição.
Accept
Define os tipos de conteúdo aceitos na resposta.

Exemplo de Resposta HTTP

            HTTP/1.1 200 OK
            Date: Wed, 25 Mar 2026 11:22:00 GMT
            Content-Type: text/html
            Content-Length: 1256

            
Bem-vindo!
Status Line
Indica versão do protocolo, código de status (200) e mensagem ("OK").
Date
Data e hora da resposta.
Content-Type
Tipo de conteúdo retornado (ex: text/html, application/json).
Content-Length
Tamanho do corpo da resposta em bytes.
Body
Conteúdo da resposta (página HTML, JSON, etc.).

Resumo

O protocolo HTTP é essencial para a Web, pois permite que navegadores solicitem recursos e servidores respondam com páginas, imagens e dados. Sem ele, não haveria comunicação estruturada entre cliente e servidor.
